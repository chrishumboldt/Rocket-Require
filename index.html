Rocket Require Module

Getting Started

Install via NPM.

npm install rocket-require

Note that this module has a dependency Rocket Tools (21kb) which will automatically be installed as well.

Start by including the necessary files.

<body>
    /* Your content goes here */
    <script src="node_modules/rocket-tools/js/tools.min.js"></script>
    <script src="node_modules/rocket-require/js/require.min.js"></script>
</body>

Full Example

In the example below we are attempting to load Bootstrap. We can see that Bootstrap has a dependency on jQuery, which has also been declared. As a result jQuery will be loaded first and once complete all the Bootstrap files will load after.

Once all files are present on the DOM a function is called displaying a message.

<script>
// Add the module information
Rocket.module.add({
   bootstrap: {
      requires: ['jquery'],
      css: [
         'https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css',
         'https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap-theme.min.css'
      ],
      js: 'https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js'
   },
   jquery: {
      js: 'https://code.jquery.com/jquery-3.1.1.js'
   }
});

// Initilise the require
var require = Rocket.require();
require.add('bootstrap');

// Load
require.load(function () {
   $('#success-text').text('jQuery and Bootstrap has just been loaded with Rocket Require!');
});
</script>
